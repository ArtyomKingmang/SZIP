## More About Library
The library allows you to create zip archives from code in the java and processing languages.

Simple example:

```java
 public void setup() {
    try {
      SZIP.archive("C:\\Users\\crowb\\OneDrive\\Рабочий стол\\example.zip", "C:\\Users\\crowb\\OneDrive\\Рабочий стол\\\\log.txt");
    }catch (IOException e) {
        e.printStackTrace();
    }
}
```
